# go-htmx-tailwind

A web app template built with:
- GO
- HTMX
- TailwindCSS

## Quick Start

Dev mode:
```bash
# Run vite in dev mode (from /frontend).
# Note that this runs `vite build --watch` under the hood.
pnpm dev

# Build and run the app (from /).
ENVIRONMENT="development" go run ./cmd/app
```
