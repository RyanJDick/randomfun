# go-htmx-tailwind

A web app template built with:
- GO
- HTMX
- TailwindCSS

## Quick Start

Build and run with:
```bash
go run ./cmd/app
```

## Decisions

- Keep the list of dependencies as small as possible.
- Structure the project code based on the "Server Project" recommendation [here](https://go.dev/doc/modules/layout)
